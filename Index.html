Método de Gauss-Jordan

Solución de sistemas de ecuaciones 3x3 y 4x4

Facultad de Ingeniería

01. Introducción

El método de Gauss-Jordan es una herramienta eficaz para resolver sistemas de ecuaciones lineales de manera sistemática. Esta calculadora automatiza el proceso para reducir errores humanos.

02. Arquitectura y Diseño

Se emplean estrategias de álgebra lineal que guían la manipulación de las celdas en Excel. La estructura cuenta con dos hojas de trabajo principales y programación en VBA.

03. Funcionamiento

El usuario selecciona la matriz e ingresa los coeficientes. Al presionar el botón "Resolver", la macro ejecuta el algoritmo de forma secuencial.

04. Resultados y Pruebas

Se validó el sistema con ecuaciones de 3x3 y 4x4, logrando soluciones exactas como x=2, y=3, z=-1 para el primer caso de prueba.
